# 📢 팀 공유: 별(Byeol, 가칭) 프로젝트 v0.3 업데이트

안녕하세요 팀. AllKpop 서브 프로젝트 **별(Byeol, 가칭)** 의 로드맵/PRD 문서를 v0.3까지 정리했습니다. 이전 "Planet" 시절(v0.1) 이후 방향이 몇 차례 조정되어, 한 번에 맥락 따라잡을 수 있도록 버전별 변경점을 아래 요약합니다.

## 🔑 지금 꼭 알아야 할 3가지

1. **프로젝트명이 "Planet"에서 "별(Byeol)"로 바뀌었고, 이는 가칭입니다.** 정식명은 MVP 베타 오픈 전 확정 예정.
2. **MVP = 베타 오픈**입니다. 그랜드 런칭이 아니라 **"게임피케이션이 체류시간을 유의미하게 올리는가"** 를 검증하는 단계. 지표 중심 운영.
3. **팬덤 기여 + 경쟁 랭킹이 MVP의 핵심 루프**입니다. 원래 Phase 4에 있던 아티스트 테마 구조를 앞으로 당겼고, IP 페이지를 9~12개 선런칭합니다 (BLINK, ARMY, MY, EXO-L, DIVE, MOA, Once, CARAT, NCTzen 등).

## 📜 버전별 변화 요약

### v0.1 → v0.2 (2026-04-14 미팅 반영)

- 프로젝트명 **Planet → 별(Byeol, 가칭)**
- **베타 오픈**으로 포지셔닝 명확화
- **팬덤 기여 구조를 Phase 4 → MVP**로 전진 (공용 별 + 아티스트 별 병행)
- **세계관 신설**: IP 부재 대응 → 씨앗/물/비료/꽃/나무 성장 메타포
- **시즌 매개체 교체** 개념 (시즌1 별 → 시즌2 섬 → 시즌3 바다)
- 미니게임 선정 기준에 **"올케이팝 IP/미디어/콘텐츠 연동 필수"** 명문화
- 게임 프로필 비밀번호 삭제 (닉네임만)
- 광고 구좌 배너/팝업 MVP 포함 명시

### v0.2 → v0.3 (팀 내부 MVP 구체화)

- **IP 페이지 규모 확대**: 2~5개 → **9~12개 선런칭** (명확한 타겟 팬덤/경쟁심리 활용)
- **온보딩 플로우 구체화**: 게임 닉네임 + **대표 팬덤 1개(필수)** + IP 다중 팔로우(선택)
- **홈 메인뷰 재정의**: 더 이상 "메인 게임"이 아니라 **별 세계 관전뷰 (3D 공전 씬)** + IP 랭킹 오버레이
- **IP 페이지 = 3D 별 키우기 공간**: 씨앗 심기 / 물 주기 / 비료 주기 + 기여자 Top 랭킹
- **미니게임 종료 UX**: 점수 획득 → 기여할 별 선택 → 랭킹 피드 포맷 확정 → **"[닉네임] [대표팬덤] N포인트를 [◯◯별]에 기여"** (경쟁심리 자극)
- **뱃지 / 메달 / 온보딩 가이드 체계는 v0.1·v0.2 그대로 유지**

## 🚀 Phase 1 베타 MVP 범위 (4/22 런칭 목표)

**High 우선순위**
- 온보딩 플로우 (닉네임 + 대표 팬덤 + IP 팔로우)
- 별 홈 관전뷰 + IP 랭킹 오버레이
- IP 페이지 (3D 별 + 씨앗/물/비료 + 기여자 랭킹)
- 미니게임 종료 후 기여 선택 + 랭킹 피드
- 게임 프로필, 인벤토리, 온보딩 가이드·메달·뱃지

**Medium**
- 일일/주간 미션, 보상 지급 로직
- 미니게임 2~3종 (IP/미디어 연동)
- 당근온도형 기여도 프로필 표시
- 배너/팝업 광고 구좌

## ❓ 팀원분들 의견 필요한 Open Questions

1. **프로젝트 정식 명칭** — "별" 그대로 갈지, 다른 네임 채택할지
2. **GA 전략** — 올케이팝 GA에 통합할지, 별 전용 GA를 둘지, 합산 리포트를 만들지
3. **저작권** — 음원·가사·기사 이미지 미니게임 활용 범위 (법무 검토)
4. **초기 IP 페이지 라인업 9~12개** 최종 확정 방식 — 편집 추천 / 사전 신청 집계 / 혼합
5. **3D 에셋 제작 방식** — 외주 / AI + 후가공 / 내부 (디자이너 부재 상황)
6. **대표 팬덤 변경 정책** — 변경 주기 제한, 쿨다운, 누적 기여도 처리
7. **공용 "올케이팝 행성"의 역할** — 관전 중심지 전용인지, 중립 기여처도 겸하는지
