# 별(Byeol, 가칭): 팬덤 기여형 게임 생태계 구축 전략

- 문서 버전: v0.3
- 작성 일시: 2026.04.15
- 이전 버전: v0.2 (2026.04.15), v0.1 (2026.04.06, "Planet")
- 상위 프로젝트: AllKpop
- **프로젝트명**: 별(Byeol) — **가칭**. 정식 명칭 미정, 오픈 전 확정 필요

> **v0.2 → v0.3 주요 변경**
> - **IP 페이지 전략 전환**: v0.2의 "사전 신청 기반 2~5개 아티스트 별" → **주요 팬덤 IP 페이지를 다수 선런칭** (BLINK, ARMY, MY, EXO-L, DIVE, MOA, Once, CARAT, NCTzen 등). "명확한 타겟 팬덤/경쟁심리"를 초기 체류 드라이버로 전면 활용
> - **프로필 진입 플로우 구체화**: 올케이팝 계정 로그인 상태 + 별 프로필 없음 → 게임용 닉네임 설정 + **대표 팬덤 1개 설정(필수)** + IP 페이지 다중 팔로우(선택)
> - **홈 화면 구조 재설계**: 상단 메인뷰는 이제 "게임"이 아니라 **"별 세계 관전뷰"**. 올케이팝 주 행성 주위를 IP 별들이 공전하는 3D 씬. 오버레이로 IP 랭킹, 클릭 시 IP 페이지 이동
> - **IP 페이지 = 3D 별 키우기 공간**으로 정의. 씨앗 심기 / 물 주기 / 비료 주기 + 해당 별 기여자 Top 랭킹 오버레이
> - **미니게임 종료 후 기여 UX 구체화**: 점수 → 기여할 별 선택 → 랭킹 표시 포맷 "[닉네임] [대표팬덤] N포인트를 [◯◯별]에 기여" (경쟁심리 자극)
> - 뱃지 / 온보딩 가이드 / 메달 구조는 v0.2 유지

## 1. 프로젝트 배경 및 전략적 방향

별(Byeol, 가칭)은 올케이팝의 "읽기" 중심 유저 경험을 "참여·체류" 중심으로 전환하기 위한 서브 프로젝트입니다. MVP는 IP 없이 막연한 팬층을 대상으로 하지 않고, **명확한 타겟 팬덤(BLINK/ARMY/MY/EXO-L/DIVE/MOA/Once/CARAT/NCTzen 등)을 선정**하여 팬덤 간 경쟁심리를 체류 드라이버로 삼습니다.

- **팬덤 중심 루프**: 개인 게임 활동이 자신이 지지하는 아티스트 별의 성장에 실시간 기여. 경쟁과 응원이 랭킹·오버레이 UI로 지속 노출
- **통합 경제 + 분기된 성취**: Sparkle / Planet EXP / Artifact는 공통. 기여·랭킹은 "아티스트 별" 단위로 분기
- **간접 기여 KPI**: 베타 단계는 체류시간·재방문·세션당 게임 수 검증
- **IP 공백 대응 세계관**: 자체 세계관(별 세계 — 행성 공전, 씨앗→물·비료→꽃·나무)으로 몰입 보완

## 2. 단계별 구축 로드맵

### Phase 1. 베타 MVP 출시: 팬덤 기여 루프 검증 (4월 22일)

**MVP 목표**: 게임피케이션이 올케이팝 체류시간을 유의미하게 올리는가 + **팬덤 기여 + 경쟁 랭킹**이 재방문을 유도하는가 검증. 베타 오픈.

#### 프로필 진입 플로우 *(v0.3 구체화)*
- 올케이팝 계정에 이미 로그인된 상태로 별 진입
- 별 전용 프로필 없음 시 온보딩 다이얼로그:
  1. **게임용 닉네임 설정** (올케이팝 계정과 별도. 별 내부 식별자)
  2. **대표 팬덤 1개 선택 (필수)** — BLINK, ARMY, MY, EXO-L, DIVE, MOA, Once, CARAT, NCTzen 등
  3. **IP 페이지 팔로우 (선택, 복수 가능)** — 대표 팬덤과 별개
- 대표 팬덤은 프로필/랭킹 표기에 기본 노출. 이후 변경 가능

#### 홈 화면 구조 *(v0.3 재설계)*
- **상단 메인뷰 = 별 세계 관전뷰**
  - v0.1/v0.2에서는 "메인 게임"이었으나, v0.3에서는 **게임이 아닌 관전·로비 공간**으로 재정의
  - 올케이팝 주 행성을 중심으로 IP 별들이 공전하는 3D 씬
  - **오버레이: IP 랭킹 Top N** — 랭킹 항목 클릭 시 해당 IP 페이지로 이동
- **IP 페이지 목록**: 존재하는 모든 IP 페이지를 리스트 또는 IP별 게임 미리보기 카드 형태로 노출
- **미니게임 목록**: 하위 섹션에 배치. 게임 썸네일 + 진입 CTA

#### IP 페이지 *(v0.3 구체화)*
- IP별로 **3D 별 오브젝트** 보유
- 유저 액션: 씨앗 심기 / 물 주기 / 비료 주기 → 별 성장 시각 변화
- 성장 경로: 미니게임 기여 + 별 키우기 액션 → 별 포인트 상승 → 홈 랭킹 순위 변동
- **오버레이: 해당 IP 기여자 Top 랭킹** (누가 이 별에 가장 많이 기여했는가)

#### 미니게임 *(v0.3 구체화)*
- 선정 기준: 올케이팝 IP/미디어/콘텐츠 연동, 짧은 학습곡선, 무한 반복 가능, 명확한 점수 산출
- 후보: 올케이팝 기사 속 보석 찾기, 케이팝 음원/가사 맞추기, 가벼운 아케이드형 1종 등
- **게임 종료 플로우**:
  1. 사용자 랭킹(이번 게임 기준) 표시
  2. "획득한 N포인트를 어떤 별에 기여할까?" 선택 → 팔로우한 IP 별 + 공용 별 중 택1 (기본값: 대표 팬덤)
  3. 선택된 별의 포인트 상승, 홈 랭킹 반영
  4. 랭킹 이벤트 로그 포맷: **"[닉네임] [대표팬덤] N포인트를 [◯◯별]에 기여"** — 경쟁심리 자극

#### 공통 보상 및 성장 체계
- Sparkle (재화) — 게임 결과 비례 지급. MVP는 적립 중심, "심기"에 소량 소비
- Planet EXP (경험치) — 개인 성장 + 별 성장
- Artifact (수집품) — 씨앗/꽃/나무 등 세계관 아이템. 특정 별에 심기 가능

#### 게임 전용 프로필 및 성취 *(v0.2에서 유지)*
- 닉네임만 (비밀번호 불필요)
- 프로필 노출: 닉네임, 대표 팬덤, 대표 뱃지, 획득 메달, 당근온도형 팬덤 기여도
- 뱃지: 신규회원 → 루키 → 파이어니어 → 익스플로러 → 커맨더
- **메달 / 온보딩 가이드 / 뱃지 체계는 v0.1·v0.2와 동일하게 유지**

#### 초기 IP 페이지 라인업 (잠정) *(v0.3 신설)*
- BLINK (BLACKPINK), ARMY (BTS), MY (Aespa), EXO-L (EXO), DIVE (IVE), MOA (TXT), Once (TWICE), CARAT (SEVENTEEN), NCTzen (NCT)
- 최종 라인업은 편집 추천 + (선택적) 사전 신청 집계로 조정. MVP에 9~12개 선런칭 목표
- 각 IP 페이지 = 동일 UI 템플릿 + 테마 컬러/로고/배경만 차별화

#### 운영 및 광고 구좌
- 이벤트 배너 + 팝업 구좌 (MVP 포함)
- "팬덤 1등 아티스트 메인 노출" 구좌는 Phase 4 고도화

## 3. 핵심 성과 지표 (KPI)

- **체류 지표 (최우선)**: 평균 세션 길이, 1인당 평균 플레이 게임 수, 포럼 대비 체류 증분
- **충성도 지표**: D1/D7 재방문율, 미션·업적 달성률, IP 팔로우 유지율
- **팬덤 기여 지표**:
  - 대표 팬덤 지정율 (온보딩 완료율)
  - IP 팔로우 평균 개수
  - 팬덤 기여 지속성 (동일 팬덤 반복 기여 비율)
  - 팬덤 밸런스 (IP별 기여 분포 편중도)
- **경쟁·랭킹 지표** *(v0.3 신설)*: 랭킹 상위 노출 후 세션 연장률, 랭킹 클릭→IP 페이지 이동률
- **경제 지표**: Sparkle Burn Rate, 광고 참여(Opt-in)
- **플랫폼 지표**: 신규 게임 추가 리드타임, 교차 플레이율, 신규 IP 페이지 추가 리드타임

## 4. 네이밍 (가칭 운영)
